template<typename T>
struct sqb::traits::IsCopyable< T >

Used to query whether a type is copyable. To make a type non-copyable specialise this class or use the macros SQBIND_DECLARE_NON_COPYABLE_CLASS_NAME or SQBIND_DECLARE_NON_COPYABLE_CLASS. Copyable types will use the assignment operator to copy them, non-copyable types by default will use memcpy and sizeof(T). To change the copy operation of a type specialise ClassTypeCopyImpl before declaring the type.
