template<typename ClassType >
ClassType* sqb::StackHandler::GetInstance ( SQInteger  index)

Gets an instance of a class from the stack without checking it's type. Use in conjunction with IsInstanceOfType.

sqb::StackHandler stack(vm);
if (!stack.IsInstanceOfType<MyClass>(1))
{
return stack.ThrowParamError(1, sqb::TypeInfo<MyClass>().m_typeName);
}
MyClass *_this = stack.GetInstance<MyClass>();
Note
MyClass must already be bound using SQBIND_DECLARE_CLASS(MyClass); and sqb::ClassDef<MyClass>.

template<typename ClassType >
ClassType* sqb::StackHandler::GetInstanceAsType ( SQInteger  index)

Gets a class instance from the stack using ClassType<T>::Get() as the typetag check. If the object at index is not an instance or does not have the correct typetag then it will return null otherwise it will return a pointer to the class instance.

sqb::StackHandler stack(vm);
MyClass* instance = stack.GetInstanceAsType<MyClass>(1);
if (instance == nullptr)
{
return stack.ThrowParamError(1, sqb::TypeInfo<MyClass>().m_typeName);
}
Note
MyClass must already be bound using SQBIND_DECLARE_CLASS(MyClass); and sqb::ClassDef<MyClass>.

template<typename ClassType >
SQInteger sqb::StackHandler::Return ( const ClassType *  instance,
SQRELEASEHOOK  hook 
)

This is an overloaded member function, provided for convenience. It differs from the above function only in what argument(s) it accepts. In some cases can be quicker than the other templated return functions as it removes the need for an extra copy constructor and also removes some extra function calls.

MyClass func();
SQInteger sqfunc_standard(HSQUIRRELVM vm)
{
return stack.Return(func());
}
SQInteger sqfunc_fast(HSQUIRRELVM vm)
{
void *memory = sq_malloc(sizeof(MyClass));
MyClass *copy = new (memory) MyClass(func());
return stack.Return(copy, &sqb::DefaultReleaseHook<sqb::IsPodType<MyClass>::kValue>::SqFree<MyClass>);
}

SQInteger sqb::StackHandler::ThrowParamError ( SQInteger  index,
const SQChar *  expectedType 
)

Throws a squirrel exception with an error message specific for wrong parameter arguments.

// squirrel code
testfunc("blah")
// c++ code inside testfunc
StackHandler stack(vm);
return stack.ThrowParamError(2, _SC("integer"));

Generates the error message "parameter 1 has an invalid type 'string' ; expected: 'integer'".

Note
index is the index on the stack not the parameter argument number so a stack index of 2 would be the first parameter to the function. Stack index 1 is the hidden parameter 0 passed to all functions.
